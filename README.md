# portfolio-analytics
Repositório de projetos de dados com foco em **operações financeiras, conciliação e análise transacional**.

Aqui não é showcase genérico de gráfico — são soluções pensadas para resolver problemas reais de negócio: inconsistência de dados, falta de visibilidade de caixa e dificuldade de tomada de decisão.

---

## O que você vai encontrar
- Consolidação de múltiplas fontes financeiras
- Modelagem de dados voltada à operação
- Conciliação de recebíveis ponta a ponta
- Análises orientadas à ação (não só visual)

---

## Stack
- SQL (PostgreSQL, Athena)
- Python
- Power BI
- AWS (S3, Glue, Lambda)
- ETL / Data Pipelines
